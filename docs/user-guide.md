# Throw In — 使い方ガイド

気になった記事・動画・PDF・メモを一か所にまとめて管理できる、個人向けデジタルライブラリアプリです。

---

## 主要機能一覧

| 機能 | 概要 |
|------|------|
| クリップ保存 | URL・PDF・YouTube・画像・テキストメモを保存 |
| AI自動処理 | 要約・タグ・カテゴリ・キーポイントを自動生成 |
| キーワード検索 | タイトル・本文・タグ・カテゴリで絞り込み |
| セマンティック検索 | 意味的に近いクリップをAIで検索 |
| コレクション | クリップをグループにまとめて整理 |
| ブックマーク | 重要なクリップにスター印をつけて後で参照 |
| 閲覧履歴 | 過去に見たクリップを時系列で確認 |
| 翻訳 | クリップの本文を任意の言語に翻訳 |
| グローバル検索 | 他ユーザーの公開クリップを横断検索 |
| フォロー | 他ユーザーをフォローして公開クリップを閲覧 |
| コメント | 公開クリップにコメント・返信 |
| インサイト | 保存クリップをもとにAIがまとめ記事を生成 |
| 通知 | フォロー・コメント・いいねを通知 |
| オフライン閲覧 | 一度読み込んだクリップはオフラインでも閲覧可 |

```
ホーム (/)            — ライブラリ一覧

ブックマーク          — ブックマーク済みクリップ
コレクション          — グループ管理
履歴                  — 閲覧履歴
通知                  — フォロー・コメント等の通知
インサイト            — AIまとめ記事

```

## 基本の使い方

### クリップを追加する

右下の **＋ボタン** をタップして保存画面を開きます。

| モード | 使い方 |
|--------|--------|
| **ウェブ記事** | URLを貼り付ける。タイトル・アイキャッチを自動取得。 |
| **YouTube動画** | YouTube の URL を貼り付ける。字幕があれば自動取得。 |
| **PDF** | ファイルをアップロード。テキスト自動抽出。 |
| **画像** | 画像URLを貼り付ける。 |
| **メモ** | テキストを直接入力。日記・メモ代わりに使える。 |

保存後、OpenAI キーが設定されていれば自動でAI処理が走ります：
- 要約（数行のサマリー）
- タグ・カテゴリ・サブカテゴリの自動付与
- キーポイント（箇条書きの要点）
- 埋め込みベクトル生成（セマンティック検索用）

### クリップを探す

**ホーム画面での絞り込み：**
- キーワード入力で即時フィルタ
- 未読のみ / ブックマーク済みのみ で表示切り替え
- カテゴリ・タグをタップして絞り込み

**検索画面：**
- キーワード検索 — タイトル・本文テキストの部分一致
- セマンティック検索（AIアイコン） — 意味が近いクリップを探す。言葉が一致しなくても見つかる。
- グローバル検索タブ — 他ユーザーの公開クリップを検索

### コレクションで整理する

コレクションはフォルダのようなもの。クリップ詳細画面からコレクションに追加できます。  
コレクション画面で一覧表示・新規作成が可能です。

## 各機能の詳細

### ブックマーク

クリップ一覧のブックマークアイコンをタップするとスターが付きます。  
ブックマーク画面ではスター付きクリップだけを一覧表示します。

### 翻訳

クリップ詳細画面から「翻訳」を実行すると、本文を任意の言語に変換します。  
デフォルト翻訳言語は設定画面で変更できます。

### 公開・グローバル検索

クリップ詳細画面で「公開する」をオンにすると、グローバル検索に表示されるようになります。  
他のユーザーが閲覧・保存・コメントできます。

### フォロー

ユーザーページからフォローできます。  
フォロー中のユーザーが公開したクリップは検索画面の「フォロー中」タブで確認できます。

### インサイト

設定した公開クリップをもとに、AI がカテゴリ別・週次などのまとめ記事を生成します。  
インサイト画面で読めます。

### 通知

以下のアクションで通知が届きます：
- 自分がフォローされた
- 公開クリップにコメントがついた
- コメントに返信がついた
- いいねされた
- 開発者からのお知らせ

通知の種類ごとにオン/オフを設定画面で切り替えられます。

## 初期設定

### OpenAI APIキーを登録する

AI機能（要約・タグ・セマンティック検索・翻訳）を使うには OpenAI APIキーが必要です。

**設定画面** → 「OpenAI APIキー」欄に入力して保存してください。  
キーはブラウザのローカルストレージにのみ保存されます。サーバーには送信されません。

### プロフィールを設定する

設定画面の「グローバル検索プロフィール」から、表示名（ニックネーム）とアバター絵文字を設定できます。  
グローバル検索で他のユーザーに表示されるため、実名は使用しないでください。

---

## AI一括処理

過去に保存済みのクリップをまとめてAI処理したい場合は、設定画面の「AI一括処理」から実行できます。

- **既存クリップを一括AI処理** — 未処理クリップに要約・タグ・カテゴリを付与
- **「その他」クリップを再分類** — カテゴリが「その他」になっているクリップを適切に振り直す

## よくある質問

**Q. 保存したデータはどこに保存されますか？**  
クラウド（Supabase）に保存されます。デバイスを変えてもログインすれば同じデータにアクセスできます。

**Q. オフラインでも使えますか？**  
一度読み込んだクリップはブラウザにキャッシュされるため、オフライン時でも閲覧できます。ただし新規保存や同期はオンライン時のみ可能です。

**Q. AI処理はすべて無料ですか？**  
AI処理はご自身の OpenAI APIキーを使用します。OpenAI の料金が別途発生します。

**Q. アカウントを削除したい**  
設定画面の一番下「アカウントと全てのデータを削除」から削除できます。削除すると全データが完全に消去され、取り消しできません。
